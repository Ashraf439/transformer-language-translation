API Reference
All endpoints are served from the same host as this app
POST /translate
Returns the best translation using beam search, along with a confidence score (0–100).
Request body
{"text": "hello , how are you ?"}
Response
{"translation": "bonjour , comment allez-vous ?", "confidence": 93.5}
POST /translate/candidates
Returns the top 3 beam search candidates ranked by score. Useful when you want to offer alternatives.
Request body
{"text": "hello , how are you ?"}
Response
{"candidates": [
  "bonjour , comment allez-vous ?",
  "bonjour , comment vas-tu ?",
  "salut , comment allez-vous ?"
]}
GET /history
Returns the last 20 saved translations ordered by most recent first.
Response
{"history": [
  {
    "english": "hello , how are you ?",
    "french": "bonjour , comment allez-vous ?",
    "confidence": 93.5,
    "timestamp": "2026-06-24 13:12:54"
  }
]}
GET /
Serves the main TransFormer web interface.
GET /history-page
Serves the translation history page (same as the History tab in the sidebar).
